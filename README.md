## Installation

Requirements:

* [VisPy](http://vispy.org/)
* [BioPython](http://biopython.org/)
* [NumPy](http://www.numpy.org/)

Install them with pip:

```
$ pip install numpy vispy biopython
```

## Running

```
$ python molecular_viewer.py data/1e57.pdb [cpk|model]
```
